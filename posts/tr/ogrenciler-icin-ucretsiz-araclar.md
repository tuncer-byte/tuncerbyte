# Aboneliklerin Kralı

## GitHub Student Developer Pack

🔗 [https://education.github.com/pack](https://education.github.com/pack)

Eğer bu listedeki **tek bir şeyi alacaksan**, o da kesinlikle bu olmalı.

GitHub Student Developer Pack aslında tek bir servis değil.

**Onlarca farklı aracın kapısını tek seferde açan bir paket.**

Avantajları:

* Tek başvuru
* Tek doğrulama
* 100+ araç aktif

Birçok öğrenci programı sadece tek bir ürün verir.

GitHub Student Pack ise **neredeyse tüm geliştirme ortamını ücretsiz kurmanı sağlar.**

Başvuru genellikle **1–3 gün içinde onaylanır.**

Başvuru için şunlardan biri yeterlidir:

* üniversite e-postası (.edu / .edu.tr)
* öğrenci kimliği fotoğrafı
* öğrenci belgesi

Paketin içindeki araçların toplam değeri **10.000 doların üzerindedir.**
