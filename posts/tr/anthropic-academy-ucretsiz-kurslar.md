13 ücretsiz kurs. Tamamlama sertifikası. Kayıt için sadece e-posta yeterli.

## Nasıl Kayıt Olunur?

1. [anthropic.skilljar.com](https://anthropic.skilljar.com/) adresine git
2. "Enroll" butonuna tıkla
3. Yeni bir Skilljar hesabı oluştur (sadece e-posta yeterli)
4. İstediğin kursa kaydol ve hemen başla

Kurslar kendi hızında ilerleyebileceğin (self-paced) formatta. Tamamlayınca sertifikayı indirip LinkedIn'e ekleyebilirsin.

## Kurslar

### Genel Kullanıcılar İçin

![Yapay zeka içerik ve yazı temsili](https://www-cdn.anthropic.com/images/4zrzovbb/website/33dbe8f783d4835a838b4c4ae85d3c04e352fee1-1000x1000.svg)

#### Claude 101

Claude'u ilk kez kullananlar için başlangıç noktası.

Ne öğrenirsin:

- İlk konuşmayı nasıl yapılandırırsın
- Prompting teknikleri
- Projects, Artifacts ve Skills özellikleri
- Rol bazlı uygulamalar (yazılım, pazarlama, eğitim vb.)

Kurs sonunda sertifika veriliyor.

---

#### AI Fluency: Framework & Foundations

Yapay zeka ile etkili, verimli ve güvenli çalışmanın çerçevesini öğretiyor.

Anthropic'in "4D Yaklaşımı" üzerine kurulu:

- **Delegation** (Görevlendirme)
- **Description** (Tanımlama)
- **Discernment** (Değerlendirme)
- **Diligence** (Özen)

Bireysel kullanım kadar ekip ve kurumsal AI okuryazarlığı için de ideal.

---

#### AI Fluency for Nonprofits

Kar amacı gütmeyen organizasyonlar için özelleştirilmiş versiyon.

Misyon odaklı çalışmalarda AI'ı nasıl kullanabileceğini anlatıyor.

### Geliştiriciler İçin

![Yapay zeka düşünce ve bağlantı temsili](https://www-cdn.anthropic.com/images/4zrzovbb/website/46e4aa7ea208ed440d5bd9e9e3a0ee66bc336ff1-1000x1000.svg)

#### Building with the Claude API

Kapsamlı bir Python kursu. Claude'u üretime almak isteyenler için.

Ne öğrenirsin:

- Kimlik doğrulama ve temel API kullanımı
- Çok turlu konuşmalar ve sistem promptları
- Tool use (araç kullanımı)
- RAG (Retrieval-Augmented Generation) pipeline'ları
- Extended thinking (uzatılmış düşünme modu)
- Görsel ve PDF işleme
- Agentic iş akışları

Ön koşul: Python ve temel JSON bilgisi.

---

#### Claude Code in Action

Claude Code'u geliştirme iş akışına entegre etmek isteyenler için.

Ne öğrenirsin:

- Claude Code'un mimari yapısı
- Bağlam yönetimi ve özel komutlar
- MCP entegrasyonu
- GitHub otomasyonu

Ön koşul: Terminal kullanımı ve temel Git.

---

#### Introduction to Agent Skills

Claude Code'da yeniden kullanılabilir "Skills" yazmak için.

Ne öğrenirsin:

- Markdown tabanlı talimat sistemi
- Bir kez öğret, defalarca kullan mantığı
- Plugin veya kurumsal ayarlar üzerinden dağıtım

---

#### Introduction to Model Context Protocol

MCP sunucu ve istemci geliştirmeye giriş.

Ne öğrenirsin:

- MCP'nin üç temel ilkeli: **Tools**, **Resources**, **Prompts**
- Python SDK ile sunucu yazımı
- Claude'u harici servislere bağlama

Ön koşul: Çalışan Python bilgisi.

Bu kurs özellikle önemli. MCP, Claude ekosisteminin giderek genişleyen altyapısı. Şu an binlerce MCP sunucusu mevcut ve bu sayı hızla artıyor.

---

#### Model Context Protocol: Advanced Topics

MCP'yi production ortamına taşımak isteyenler için.

Ne öğrenirsin:

- Sampling ve loglama
- Dosya erişim kontrolü
- Transport mekanizmaları (HTTP/SSE)
- Ölçeklendirme ve güvenlik

Ön koşul: Async Python ve HTTP/SSE deneyimi.
