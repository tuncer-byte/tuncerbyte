In March 2026, Anthropic launched [Anthropic Academy](https://anthropic.skilljar.com/).

13 free courses. Completion certificates. Only an email address required to sign up.

No Claude subscription or existing Anthropic account needed.

---

## How to Enroll

1. Go to [anthropic.skilljar.com](https://anthropic.skilljar.com/)
2. Click "Enroll" on any course
3. Create a free Skilljar account (email only)
4. Start learning immediately

All courses are self-paced. When you finish, you can download your certificate and add it to LinkedIn.

The starting point for anyone new to Claude.

- How to structure your first conversations

#### AI Fluency: Framework & Foundations

Teaches a framework for working with AI effectively, efficiently, ethically, and safely.

Built around Anthropic's **4D Approach**:

- **Delegation** — what to hand off to AI
- **Description** — how to communicate clearly
- **Discernment** — evaluating AI output critically
- **Diligence** — maintaining responsibility

Great for individual users and teams building organizational AI literacy.

[→ Enroll](https://anthropic.skilljar.com/)

### For Developers

![AI thinking and connection representation](https://www-cdn.anthropic.com/images/4zrzovbb/website/46e4aa7ea208ed440d5bd9e9e3a0ee66bc336ff1-1000x1000.svg)

#### Building with the Claude API

A comprehensive Python course for shipping Claude in production.

What you learn:

- Authentication and core API usage
- Multi-turn conversations and system prompts
- Tool use
- RAG (Retrieval-Augmented Generation) pipelines
- Extended thinking mode
- Image and PDF processing
- Agentic workflows

Prerequisites: Python proficiency, basic JSON knowledge.

[→ Enroll](https://anthropic.skilljar.com/)

#### Claude Code in Action

For developers integrating Claude Code into their workflow.

What you learn:

- Claude Code's architecture and context management
- Custom commands
- MCP integration
- GitHub automation

Prerequisites: Terminal usage, basic Git.

[→ Enroll](https://anthropic.skilljar.com/)

#### Introduction to Agent Skills

How to build reusable Skills in Claude Code.

What you learn:

- Markdown-based instruction system
- Write once, reuse everywhere
- Distribution via plugins or enterprise settings

[→ Enroll](https://anthropic.skilljar.com/)

---

#### Introduction to Model Context Protocol

Building MCP servers and clients from scratch.

What you learn:

- MCP's three core primitives: **Tools**, **Resources**, **Prompts**
- Writing servers with the Python SDK
- Connecting Claude to external services

Prerequisites: Working Python knowledge.

This course is particularly relevant right now. MCP is becoming the standard infrastructure layer for the Claude ecosystem — thousands of servers exist and the number is growing fast.

[→ Enroll](https://anthropic.skilljar.com/)

#### Model Context Protocol: Advanced Topics

Taking MCP to production.

What you learn:

- Sampling and logging
- File access control

- Scaling and security patterns

Prerequisites: Async Python, HTTP/SSE familiarity.

[→ Enroll](https://anthropic.skilljar.com/)
